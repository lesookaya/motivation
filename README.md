# Salary Calculator\
Калькулятор мотивации сотрудников на основе KPI и воронки продаж.\
***
## **Возможности**\
* расчет KPI по весам\
* расчет премии\
* расчет итоговой зарплаты\
* добавление этапов воронки\
* сохранение настроек в браузере\
***
## **Использование**\
1. Укажите оклад\
2. Укажите максимальную премию\
3. Заполните план и факт по каждому KPI\
4. Система автоматически рассчитает премию\
