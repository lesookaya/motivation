## Salary Calculator
Калькулятор мотивации сотрудников на основе KPI и воронки продаж.
# Возможности
расчет KPI по весам
расчет премии
расчет итоговой зарплаты
добавление этапов воронки
сохранение настроек в браузере
# Использование
Укажите оклад
Укажите максимальную премию
Заполните план и факт по каждому KPI
Система автоматически рассчитает премию
